# Repository overview
The basic dino game in Godot was created for school purposes. Your task is simple - collect all coins through the level to get to another level.

# Keyboard controls 
| KEY  | ACTION |
| ------------- | ------------- |
| D / right arrow | Move right  |
| A / left arrow | Move left  |
| W / up arrow | Jump  |
| ESC | Quit game  |
